# ગિટલેબ પેજીસ (GitLab Pages) માટેનું કન્ફિગરેશન pages: # આ સ્ટેજ નક્કી કરે છે કે આ ડિપ્લોયમેન્ટ (Deployment) પ્રોસેસ છે stage: deploy # આ સ્ક્રિપ્ટ તમારી ફાઇલોને ગિટલેબના સર્વર પર સેટ કરશે script: - mkdir .public - cp -r * .public/ - mv .public public # આ આર્ટિફેક્ટ્સ (Artifacts) ગિટલેબને જણાવે છે કે કયું ફોલ્ડર પબ્લિક કરવાનું છે artifacts: paths: - public # આ નિયમ નક્કી કરે છે કે જ્યારે તમે 'main' બ્રાન્ચમાં કોડ પુશ કરશો ત્યારે જ વેબસાઈટ લાઈવ થશે only: - main
